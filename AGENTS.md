## Pull Requests

When creating automatic PRs:

1. Check for `.github/PULL_REQUEST_TEMPLATE.md` in the repository
2. If template exists, read it and follow its structure exactly
3. Add summary and changes within the `## Description` section of the template
4. Fill in all required sections from the template
5. Do not skip or omit any sections - use "N/A" if a section doesn't apply
